# System Interface

System Interface is a Solo Leveling inspired personal management system that helps you stay on top of quests, penalties, and progress tracking.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Building for Production

```sh
npm run build
```

The build output is generated in the `dist` directory and can be deployed to any static hosting service such as Vercel, Netlify, or Cloudflare Pages.
